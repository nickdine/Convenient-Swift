##How to build
1)  Clone the repository
```
$ git clone https://github.com/xuzhou524/Convenient-Swift.git
```
2) Open the workspace in Xcode
```
$ open "Convenient-Swift.xcworkspace"
```
3) Compile and run the app in your simulator or iOS device
